# Lunar 🌙

Aplicativo iOS desenvolvido durante o programa **Apple Developer Academy**, com foco em uma experiência de navegação lunar interativa e imersiva.

## 📱 Sobre o Projeto

Lunar é um aplicativo que simula a exploração da superfície lunar, combinando design visual cuidadoso com animações fluidas. Desenvolvido como projeto de aprendizado no Apple Developer Academy CIn-UFPE.

## 🛠 Tecnologias

- **Swift**
- **SwiftUI**
- **Animations & Transitions**
- **Xcode**

## ✨ Funcionalidades

- Interface imersiva com tema lunar
- Animações e transições personalizadas
- Navegação interativa entre telas
- Design responsivo para diferentes telas iOS

## 🏫 Contexto

Projeto desenvolvido no **Apple Developer Academy** — CIn-UFPE (Recife, PE, Brasil).
