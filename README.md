## Sobre o Projeto

**Lunar** é um aplicativo iOS que simula a exploração da superfície lunar, combinando design visual cuidadoso com animações fluidas. Desenvolvido como projeto de aprendizado no Apple Developer Academy CIn-UFPE.

---

## Tecnologias

- **Swift** — linguagem principal do desenvolvimento
- **SwiftUI** — construção da interface e animações
- **Animations & Transitions** — animações e transições personalizadas
- **Xcode** — ambiente de desenvolvimento

---

## Funcionalidades

- Interface imersiva com tema lunar
- Animações e transições personalizadas
- Navegação interativa entre telas
- Design responsivo para diferentes telas iOS
